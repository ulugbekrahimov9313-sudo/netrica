Frontend va Backend Farqi Nima?

Dasturlashni o`rganmoqchi bo`lganlar ko`pincha savol beradi: Frontend yoki Backend? Bu maqolada ikkalasining farqini tushunarli tilda tushuntiramiz.

Frontend Nima?

Frontend ? bu foydalanuvchi ko`radigan qism. Saytning dizayni, tugmalar, rasmlar ? bularning barchasi Frontend.

Frontend texnologiyalar: HTML, CSS, JavaScript, React, Vue.js

Backend Nima?

Backend ? bu saytning ichki qismi. Ma`lumotlar bazasi, server, API ? bularning barchasi Backend.

Backend texnologiyalar: Python, Node.js, PHP, Java, PostgreSQL

Qaysi Birini O`rganish Kerak?

Bu sizning qiziqishingizga bog`liq:

Dizayn va ko`rinish yoqsa ? Frontend

Mantiq va tizim yoqsa ? Backend

Ikkalasi ham qiziq bo`lsa ? Fullstack

Fullstack dasturchi ? Frontend va Backend ikkalasini ham biladi.
